# 🚴 Load Profile Monitoring System for E-Cycles

An IoT-based monitoring system designed to analyze the power consumption and battery performance of electric cycles under varying riding conditions. The system uses multiple sensors to monitor environmental and operational parameters, helping improve battery efficiency and optimize energy usage.

---

## 📖 Overview

The project studies how different operating conditions such as terrain, rider weight, and ambient temperature affect an electric cycle's battery discharge and motor load.

Using an ESP8266 microcontroller with multiple sensors, the system collects real-time data and displays it through a cloud-connected monitoring platform.

---

## ✨ Features

- 🔋 Battery load monitoring
- 📈 Load profile analysis
- 🌡 Temperature monitoring
- ⛰ Slope detection
- ⚖ Rider load analysis
- 📱 Blynk cloud monitoring
- 📊 Real-time sensor data visualization
